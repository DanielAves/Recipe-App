@author Daniel Aves, Matthew Cutts, Dominic Heyhoe

This app is a food recipe application which allows users to view and filter recipes obtained from the API supplied at -https://www.edamam.com/.
The features of the app are as follows: 

- Ability to search for a recipe based on a keyword such as 'Chicken' 
- Filters which permit advanced searching based on health requirements, cuisine type, meal plan, number of results etc. 
- Facility to search based on ingredients alone. 
- Shopping list utility to store items for your shopping trip 
- Conversion screen to convert between the variation of common metrics often found in recipes. 


app/src/main - Contain the main project code
app/src/androidTest - Contains Espresso instrumented tests
app/src/test - Contains unit tests
